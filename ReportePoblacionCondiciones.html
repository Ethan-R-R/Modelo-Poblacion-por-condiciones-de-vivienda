Primer Modelo

Primero se va a generar un modelo con todas las variables, solo para tener una referencia cuando los vayamos a comparar ms adelante. Se separarn los datos de prueba y datos de entrenamiento, adems de datos de prueba internas al entrenamiento para evitar fugas de datos.

Para este modelo de regresin se utilizar la funcin OLS de la libreria statsmodels.api. OLS significa ordinary least squares y en esta metodologa se busca reducir una mtrica, usualmente el Residual Sum of Squares en los coeficientes de un modelo lineal de multiples variables.

Esta librera es muy prctica porque nos puede regresar un resumen con todos los coeficientes y su valor p- ante una prueba de hiptesis con el estadstico t. Esto nos indicara si los coeficientes encontradaos de cada variable del modelo de regresin es estadsticamente significativo.

Tambin se incluye una prueba de hiptesis para un estadstico F que prueba si al menos una variable es estadsticamente significativa (para evitar falsos descubrimientos).

Adems, con los "datos de preba internos" se calcular la R^2 ajustada y RSE; para acelerar el proceso la R^2 se encontrar con la funcin r2_score de la librera sklearn.metrics. Aunque el resumen de statsmodels.api si nos da una R^2 y R^2 ajustada, se calcula con los datos de entrenamiento, lo que hace que el modelo pueda tener un sobreajuste y aparentar tener valores ms altos de los que debera.
